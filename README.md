# LedgerPro Finance Dashboard

A modern business finance dashboard built with HTML, CSS, JavaScript, and Chart.js. The interface is designed to feel like a polished SaaS product for finance teams, with executive KPIs, analytics charts, budget tracking, activity updates, and product performance insights.

## Overview

LedgerPro gives finance operators a clean single-page view of business performance. The dashboard highlights budget health, revenue growth, outstanding invoice risk, cash flow movement, recent financial activity, and top product revenue.

The project is intentionally lightweight and can run directly in the browser without a build step.

## Features

- Responsive SaaS-style application shell
- Modern sidebar with active states and mobile drawer behavior
- Sticky top navigation with search, notifications, date range, and profile menu
- Executive KPI cards with trend badges and sparkline charts
- Revenue trend line chart
- Budget vs actual spending bar chart
- Cash flow area chart
- Sortable budget overview table with sticky header
- Status badges for budget health
- Activity feed for payments, invoice updates, and budget alerts
- Top products section with progress bars and trend indicators
- Accessible color system using primary and semantic status colors
